Nuen Motor Sona ECU Board with STM32F412RGTx LQFP64 MCU
======================================================

For more details [visit Nuen Motor](https://nuenmoto.com/en).

## Flashing the kernel

The kernel can be programmed using Segger Jlink. `cd` into `boards/discovery_f412g`
directory and run:

```bash
$ make flash

(or)

$ make flash-debug
```
